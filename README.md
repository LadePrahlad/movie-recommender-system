# 🎬 Movie Recommendation System

## 🚀 Overview
This project is an advanced **Movie Recommendation System** built using multiple machine learning techniques.  
It provides personalized movie suggestions using:

- 🔥 Weighted Collaborative Filtering  
- 🎯 Content-Based Filtering (TF-IDF)  
- 🤝 Hybrid Recommendation System  
- 📊 KNN (Item-Based Filtering)  

The system is deployed using **Streamlit** for an interactive user interface.

---

## 🧠 Key Features

✅ Personalized recommendations based on user behavior  
✅ Handles **cold-start problem** (new users)  
✅ Combines multiple recommendation techniques (Hybrid Model)  
✅ Optimized using **sparse matrices** for efficiency  
✅ Fast inference using **pre-trained models (pickle)**  
✅ Interactive UI using Streamlit
